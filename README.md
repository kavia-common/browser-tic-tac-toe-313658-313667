# Browser Tic Tac Toe (React)

This repository contains a React frontend for a browser-based Tic Tac Toe experience. The intended game is a classic 3x3, two-player, turn-based Tic Tac Toe with win/draw detection.

The frontend lives in `tic_tac_toe_frontend/`.

## Where to start

- Frontend app: `tic_tac_toe_frontend/`
- Frontend documentation (setup, scripts, env vars): `tic_tac_toe_frontend/README.md`

## Preview note

The app is typically served via the preview system on port 3000. Use the preview URL provided by the environment rather than manually starting services in the preview environment.
